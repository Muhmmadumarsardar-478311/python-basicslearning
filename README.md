# Python Basics Learning

This repository contains beginner-friendly Python programs and exercises to help you learn Python basics.  
It covers topics like variables, loops, functions, conditional statements, basic OOP, and simple Python projects.

---

## Topics Covered
- Variables and Data Types
- Loops (`for`, `while`)
- Conditional Statements (`if`, `else`, `elif`)
- Functions
- Basic Object-Oriented Programming (OOP)
- Simple Python projects for practice

## How to Run

1. **Clone the repository** to your computer:
git clone https://github.com/Muhammadumarsardar-478311/python-basiclearning.git

2. Go to the repository folder:
cd python-basiclearning

3. Run any Python file:
python filename.py

Replace filename.py with the Python file you want to run.
